EmojiWorld 점검 요약

이 프로젝트는 Next.js 기반으로 개발되었으나, 현재 로컬에는 단순 버전만 있습니다.

핵심 이슈

  • Next.js 개발 서버는 npm run dev로 실행 (포트 3001)
  • /_next/static/* 404 에러 → 잘못된 포트로 접속
  • 루트 index.html이 0바이트 → 정적 서빙 시 빈 화면
  • Firebase Hosting은 out/ 폴더 서빙 필요
  • scripts/firebase-startup.sh 파일 누락

해결 방법

  • 개발: npm run dev 실행 후 표시된 로컬 주소로 접속
  • 배포: npm run build 후 out/ 폴더 서빙
